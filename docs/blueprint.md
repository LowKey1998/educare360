# **App Name**: EduCare360

## Core Features:

- Secure User Authentication & Authorization: Robust user authentication (Admin, Teacher, Parent roles) using Firebase Authentication, integrated with role-based access control.
- Admissions & Enrollment Management: Streamlined process for managing student applications, enrollment, and storing essential admission documents within Firebase Realtime Database.
- Pupil Information Management: Centralized system to maintain and update student profiles, parent contact details, and vital medical records in Firebase Realtime Database.
- Attendance Tracking: Teachers can record daily student attendance, and school administrators can monitor attendance logs and summaries via Firebase Realtime Database.
- Parent Engagement Portal: A secure read-only portal for parents to view their child's attendance records, enrollment status, and other key information stored in Firebase Realtime Database.
- Admin Dashboard & Basic Reporting: A centralized dashboard for school administrators to visualize key metrics like total enrollment, daily attendance summaries, and pending applications fetched from Firebase Realtime Database.
- AI-powered Communication Draft Tool: An intelligent tool that assists teachers and administrators in generating draft communications (e.g., newsletters, attendance notifications) using selected data from the system.

## Style Guidelines:

- Color scheme: Dark theme, suggesting professionalism and focus, derived from a concept of intelligent growth and digital clarity. This decision was informed by the app's use in educational administration, where clarity and professional presentation are key.
- Primary color (active elements, accents): Vibrant Blue (#2680D9). This hue evokes trust, intelligence, and a modern technological feel, contrasting effectively against a dark background.
- Background color: Dark Slate (#21262C). This provides a deep, understated foundation, subtly integrating the primary blue for cohesion while maintaining a clean look.
- Accent color (highlights, notifications): Bright Cyan (#85E0E0). Analogous to the primary, it's notably brighter and more saturated, designed to draw attention to critical information or interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and objective appearance, suitable for displaying data and administrative text clearly across various UI elements.
- Icons should be minimalist and line-based, using simple, clear visuals to represent modules and actions without visual clutter.
- Dashboard-centric layout featuring a persistent left sidebar for main navigation and a primary content area structured with cards for organized data display.
- Subtle, smooth animations for navigation transitions, element hovers, and data updates, enhancing the user experience without being distracting.
